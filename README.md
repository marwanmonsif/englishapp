# EduEnglish — English Learning Platform

A full-stack LMS for an English teacher in Egypt, built with Flask + SocketIO.

---

## 🚀 Quick Start

### 1. Install dependencies
```bash
pip install -r requirements.txt
```

### 2. Run the app
```bash
python app.py
```

### 3. Open in browser
```
http://localhost:5000
```

## 🗄️ Database

SQLite in development (`instance/englishapp.db`).  
To migrate to PostgreSQL, change `SQLALCHEMY_DATABASE_URI` in `app/__init__.py`:
```python
app.config['SQLALCHEMY_DATABASE_URI'] = 'postgresql://user:pass@host/dbname'
```

---

## 💬 Chat System

- Real-time via Flask-SocketIO
- Separate room per grade
- Bad words are filtered automatically (replaced with ***)
- Messages with 3+ violations are auto-deleted
- Students can report messages; teacher reviews reports

---

## 🔒 Subscription Codes

Format: `ENG-XXXX-YYYY`  
Generated by teacher dashboard → Codes page.  
Each code is tied to a specific grade and has an expiry date.
